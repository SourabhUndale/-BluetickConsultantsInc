# User Management App

This is a simple User Management Application built with React that allows users to view, add, edit, and delete user information. The user data includes fields such as first name, last name, username, age, marital status, employment status and founder status.

## Features

* View a list of users.
* Add new users.
* Edit existing users.
* Delete users from the list.
* Fetches data from a mock API for demonstration purposes.

### Technologies Used

* React: Frontend UI framework.
* TypeScript: For static type-checking and better development experience.
* CSS: For styling components.
